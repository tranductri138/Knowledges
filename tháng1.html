Tháng 1 – Làm quen AI, Prompt Engineering & Chatbot nội bộ

🎯 Mục tiêu tháng

  • Hiểu cơ bản về AI, LLM, Embedding, RAG.
  • Thành thạo Prompt Engineering cơ bản & nâng cao.
  • Biết gọi API AI (OpenAI, Ollama).
  • Biết dùng Vector DB (Qdrant) để tìm kiếm thông minh.
  • Xây dựng chatbot nội bộ có thể trả lời dựa trên tài liệu công ty.

📅 Tuần 1 – LLM cơ bản & Gọi API

  • Mục tiêu: Hiểu LLM là gì, cách hoạt động, và gọi API AI cơ bản.
  • Nội dung học:
    • Khái niệm: AI, ML, DL, LLM.
    • Transformer, token, context window.
    • OpenAI API (Chat Completion, Embedding).
    • Cài môi trường Node.js + Python cho AI.
  • Thực hành:
    1. Tạo tài khoản OpenAI, lấy API key.
    2. Viết script Node.js gọi GPT-4o để trả lời câu hỏi.
    3. Viết script Python gọi GPT-4o để tóm tắt văn bản.
  • Kết quả cuối tuần: Gọi API GPT-4o từ Node.js và Python thành công.
  • Tài nguyên: OpenAI API Docs

📅 Tuần 1.5 – Prompt Engineering

  • Mục tiêu: Viết prompt hiệu quả để điều khiển LLM.
  • Kỹ thuật:
    • Zero-shot
    • Few-shot
    • Chain-of-Thought (CoT)
    • Role Prompting
    • Output Formatting
  • Thực hành:
    1. Viết prompt zero-shot để tóm tắt văn bản.
    2. Viết prompt few-shot để phân loại email spam.
    3. Viết prompt role-based để AI đóng vai backend dev tư vấn kiến trúc.
    4. Viết prompt yêu cầu output JSON và parse trong Node.js.
  • Kết quả cuối tuần: Có bộ prompt mẫu tái sử dụng.
  • Tài nguyên: Prompt Engineering Guide

📅 Tuần 2 – Vector Database & Semantic Search

  • Mục tiêu: Hiểu Embedding và tìm kiếm thông minh.
  • Nội dung học:
    • Embedding (biểu diễn văn bản thành vector).
    • Qdrant (Vector Database).
    • Semantic Search (tìm kiếm theo ý nghĩa).
  • Thực hành:
    1. Cài Qdrant local (Docker).
    2. Dùng OpenAI Embedding API để tạo vector cho tài liệu.
    3. Lưu vector vào Qdrant.
    4. Viết API Node.js tìm kiếm tài liệu theo câu hỏi.
  • Kết quả cuối tuần: API tìm kiếm thông minh hoạt động.
  • Tài nguyên: Qdrant Docs

📅 Tuần 3 – Chạy LLM Local với Ollama

  • Mục tiêu: Chạy LLM open-source trên máy.
  • Nội dung học:
    • Ollama (chạy LLaMA 3, Mistral, Qwen local).
    • So sánh LLM cloud vs local.
    • Gọi Ollama API từ Node.js.
  • Thực hành:
    1. Cài Ollama.
    2. Tải model LLaMA 3 hoặc Mistral.
    3. Viết API Node.js gọi Ollama để trả lời câu hỏi.
    4. So sánh tốc độ và chi phí với OpenAI API.
  • Kết quả cuối tuần: Chạy LLM local thành công.
  • Tài nguyên: Ollama Docs

📅 Tuần 4 – Chatbot Nội Bộ (Mini Project)

  • Mục tiêu: Kết hợp kiến thức để làm chatbot RAG thông minh.
  • Pipeline: User query → Embedding → Search Vector DB → LLM trả lời.
  • Thực hành:
    1. Chuẩn bị tài liệu công ty (PDF, TXT).
    2. Chia nhỏ tài liệu → tạo embedding → lưu vào Qdrant.
    3. Viết API Node.js:
      • Nhận câu hỏi từ user.
      • Tìm tài liệu liên quan trong Qdrant.
      • Gửi tài liệu + câu hỏi vào LLM (OpenAI hoặc Ollama) với prompt role-based.
    4. Deploy API lên server Linux.
  • Kết quả cuối tuần: Chatbot nội bộ trả lời dựa trên tài liệu công ty.
  • Tài nguyên: RAG Concept
